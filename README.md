# 日本旅行计划管理

一个简洁优雅的旅行计划管理工具，帮助你整理行程、行李清单、想去的地方、酒店信息以及各种备注。

## 功能特性

### 🗓️ 行程概览
- 按日期展示每日行程
- 支持添加活动安排（时间 + 内容）
- 记录当日备注和注意事项
- 出行倒计时显示

### 🎒 行李清单
- 预置常用分类（衣物、洗漱、电子设备、证件、药品、杂物）
- 支持自定义添加分类和物品
- 勾选已完成打包的物品
- 一键重置清单

### 📍 目的地 & 餐厅
- 记录想去的地方
- 支持景点/餐厅/购物三种类型
- 评分、地址、备注、图片
- 标记「已去过」

### 🏨 酒店
- 展示行程中的酒店
- 记录入住/退房日期、地址、交通提示
- 支持联系方式和预订备注

### 📝 杂项备注
- 自由分类的笔记区域
- 预置签证、货币、保险等常用分类

### 💾 数据管理
- 数据自动保存到浏览器本地存储
- 支持导出 JSON 数据备份
- 支持导入 JSON 数据恢复

## 使用方式

直接在浏览器中打开 `index.html` 即可使用。

```bash
# macOS
open index.html

# Windows
start index.html

# Linux
xdg-open index.html
```

## 界面预览

- **桌面端**：左侧导航栏 + 右侧内容区
- **移动端**：底部 Tab 导航，单列卡片布局

## 技术栈

- 纯 HTML5 + CSS3 + Vanilla JavaScript
- 无需后端，数据存储在浏览器 LocalStorage
- Google Fonts（Noto Sans JP / Noto Serif JP）

## 文件结构


## 注意事项

- 图片使用 URL 方式加载，建议使用稳定的图床链接
- 建议定期导出数据备份
- 清除浏览器数据会导致本地存储丢失
